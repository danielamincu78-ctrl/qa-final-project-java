# qa-final-project-java
Proiect final 

![CI](https://github.com/danielamincu78-ctrl/qa-final-project-java/actions/workflows/ci.yml/badge.svg)

Acesta va fi proiectul final al cursului de testare automtata. 
Pentru rularea locala a testelor se descarca proiectul din git, folosind git pull. 

Docker este folosit pentru a rula testele într-un mediu curat și identic, fara sa mai intampinam problema clasica “merge pe laptopul meu”

Se adauaga in proiectul final implementarea unui mic program Java care utilizează concepte OOP (clase, moștenire, metode) și gestiunea excepțiilor. 
Clasele adaugate sunt urmatoarele: 
- InvalidUserDataException.java: semnaleaza date invalide la crearea unui utilizator, 
- User.java: defineste proprietatile utilizatorilor, 
- AdminUser.java: clasa ce mosteneste proprietatile de la User si adauga o noua proprietate privata, 
- UserRepository.java: adauaga un utilizator in lista, 
- Main.java: este clasa principala pentru testarea aplicatiei
